Data Assets API

Data Assets REST API is used by the Assets management web application to integrate the front end UI. This API shall be used by external partners to integrate xplore.io discovery engine.

Secured API

By utilizing Identity Management system using OAuth 2.0 authentication, we provider secured API for use by authenticated consumers only. Any API consumer shall be provided with security token to be used in the HTTP header. Secured token also used for:

  • Role based security for authorization
  • Log user actions for audit log
  • Social login support

RESTFull API

The API follows standard RESTFull guidelines meaning every domain entity is treated like a resource and CRUD operations shall be applied using HTTP verbs.

  • GET to fetch single or list of domain entities
  • POST to create a new domain entity or perform specific action
  • PUT to update domain entity
  • DELETE to delete domain entity

Integrations
